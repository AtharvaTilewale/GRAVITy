Get Started with the CLI

Install the command-line tool on your local machine in just three steps.

# 1. Clone the repository from GitHub
git clone https://github.com/AtharvaTilewale/GRAVITy

# 2. Navigate and run the setup script
cd GRAVITy
bash setup

# 3. Execute the program
gravity
